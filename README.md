# 🪨📄✂️ Pedra, Papel e Tesoura - Android

![Kotlin](https://img.shields.io/badge/Kotlin-Language-purple?logo=kotlin)

Um aplicativo simples e divertido do clássico jogo **Pedra, Papel e Tesoura**, desenvolvido para Android utilizando **Jetpack Compose** e **Kotlin**.

O projeto foi criado com o objetivo de praticar **lógica de programação, gerenciamento de estado e desenvolvimento de interfaces**.

---

# 📱 Sobre o aplicativo

Neste jogo, o usuário escolhe entre:

* 🪨 Pedra
* 📄 Papel
* ✂️ Tesoura

O aplicativo então gera automaticamente a escolha do computador e determina o resultado da rodada.

O resultado pode ser:

* ✅ Vitória
* ❌ Derrota
* 🤝 Empate

A interface foi construída utilizando **Jetpack Compose**, oferecendo uma UI moderna e reativa.

---

# 🚀 Tecnologias utilizadas

* Kotlin
* Jetpack Compose
* Material Design
* Android Studio
* State Management com Compose

---

# 🎮 Funcionalidades

* Seleção entre Pedra, Papel ou Tesoura
* Escolha automática do computador
* Exibição do resultado da rodada
* Alteração visual da interface conforme o resultado
* Interface moderna utilizando Compose
